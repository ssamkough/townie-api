# Townie API 🏠

API for Townie.

## Prerequisites 🗒️

Install [Docker](https://docs.docker.com/get-docker/).

Once repo pulled:
`docker-compose up`

## Technologies 🖲️

- [Node.js](https://nodejs.org/en/)
- [Express.js](https://expressjs.com/)
